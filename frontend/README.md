# Frontend - Profinance Nexus

A high-performance React application for managing project finance documents and AI synthesis.

## Core Technologies
- **React 19** with **TypeScript** for robust development.
- **Vite** for fast HMR and optimized builds.
- **Zustand** for lightweight, predictable state management.
- **Tailwind CSS** for modern, responsive styling.
- **Framer Motion** for premium interactive animations.

## Key Directories
- **src/api/**: API clients and interceptors for the backend.
- **src/components/**: Reusable UI components.
- **src/views/**: Top-level page components (Dashboard, IM Synthesis, etc.).
- **src/store/**: Zustand state stores.

## Development Workflows
1. **Ensure backend is running**.
2. **Install dependencies**: `npm install`
3. **Run locally**: `npm run dev`
4. **Build for production**: `npm run build`

## Design System
The frontend follows a premium, institutional design aesthetic using specialized color palettes, monospaced metrics, and subtle micro-animations to enhance user trust and focus.
